# 外反母趾

!!! info "本ページは準備中（骨子のみ）"

## どんな病気？

足の親指（母趾）の付け根が **「く」の字** に曲がって、内側の骨が飛び出す変形です。
歩くと痛い、靴が履きにくい、見た目が気になる、などの症状があります。

## 原因

- 女性に多い（ホルモン、関節のやわらかさ）
- 家族歴
- **先の細い靴・ハイヒール** の長期使用
- 偏平足、関節リウマチ など

## 重症度（角度で評価）

| 程度 | 外反母趾角 |
|------|------|
| 軽度 | 15〜25度 |
| 中等度 | 25〜40度 |
| 重度 | 40度以上 |

## 治療

### 1. まずは保存治療

- **靴の見直し**（先の広い、ヒールの低いもの）
- **装具・サポーター**（夜間スプリント、トゥセパレーター）
- **足の指の運動**（タオルギャザー、足の指じゃんけん）
- 鎮痛薬

### 2. 手術

- 痛みが強く、日常生活に支障があるとき
- 多くの種類があり、変形の程度で術式を選びます
- 一般に **骨を切って向きを変える「骨切り術」** が中心
- 重度では **関節を固定する** 手術もあります

### 3. 手術後の生活

- 多くは **前足部ヒール靴**（かかとで歩く特殊な靴）を 4〜6週
- 入院は短期（数日〜1週間）が多い
- デスクワーク復帰: 2〜4週
- 立ち仕事復帰: 6〜8週
- スポーツ復帰: 3〜6か月

## 注意点

- **見た目だけ** での手術は基本的に行いません
- 再発を防ぐため、術後も **靴選びと足の運動** を続けることが大切
- 一度に両足を手術するかは、生活状況と相談して決めます
