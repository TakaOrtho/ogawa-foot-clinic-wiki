# 外反母趾：後療法

!!! info "骨子（要拡張）"

## 1. 周術期管理の特徴

- 多くが **日帰り〜短期入院**（術式・施設による）
- 中足骨骨切り術後は **荷重制限**（術式により異なる）
- **前足部ヒール靴**（forefoot off-loading shoe）を 4–6 週装着
- 抗血栓薬は短期予防が中心（リスク群のみ）

## 2. 観察項目

| 項目 | チェック |
|------|----------|
| 創部 | 内側縦切開 or MIS の小切開、滲出・腫脹 |
| 母趾循環・感覚 | **深腓骨神経**（第1-2趾間）、**背側内側神経**（母趾内側） |
| 腫脹 | 中足骨骨切り後は遷延しやすい（数か月） |
| 装具・テーピング | 内反位戻り防止、適切な装着 |
| 疼痛 | NRS、薬効、リハ前予防鎮痛 |

## 3. 術後プロトコル（標準）

| 時期 | 荷重 | 装具 | リハビリ |
|------|------|------|---------|
| 0–2 週 | **踵荷重のみ**（前足部ヒール靴） | ヒール靴 + 内反テープ | 浮腫管理、足趾自動運動、近位筋維持 |
| 2–6 週 | 踵荷重継続、徐々に拇趾も | ヒール靴継続 | MTP 受動的可動域、内在筋訓練 |
| 6–12 週 | 通常靴へ移行 | 通常靴 + 装具 | 抵抗運動、バランス、歩行再学習 |
| 3 か月〜 | フリー | — | スポーツ復帰（術式・骨癒合に応じて） |

## 4. リハビリ重点項目

- **拇趾 MTP の屈曲・伸展可動域** 確保（拘縮予防が最重要）
- **内在筋トレーニング**（short foot, toe spread, towel curl）
- **歩行再学習**（拇趾蹴り出しの再獲得）
- **後足部・体幹の連動評価**
- アキレス腱・腓腹筋ストレッチ

## 5. 退院指導

- 入浴・シャワー（抜糸後）
- 前足部ヒール靴の正しい使用（踵から接地）
- 内反テーピングの自己管理
- 母趾を **強く曲げない・蹴り出さない** 期間（術式に応じ）
- 危険サイン（強い痛み・しびれ・冷感・発熱・創部発赤）

## 6. 復職目安

| 仕事 | 復帰目安 |
|------|----------|
| デスクワーク | 2–4 週 |
| 立ち仕事・軽労働 | 6–8 週 |
| 重労働・スポーツ | 3–6 か月 |
| ハイヒール着用 | 推奨しない（再発リスク） |

## 7. 高齢患者の留意点

- 転倒リスク（前足部ヒール靴の不安定性）
- 認知・せん妄
- 骨癒合遅延（糖尿病・骨粗鬆症）
- 通院困難 → 訪問看護・通所リハ調整
