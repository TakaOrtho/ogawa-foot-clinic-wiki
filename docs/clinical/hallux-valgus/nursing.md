# 看護師向け：外反母趾

!!! info "本ページは準備中（骨子のみ）"

## 周術期の特徴

- 多くが日帰り〜短期入院（術式・施設による）
- 中足骨骨切り術が中心 → **荷重制限**（術式により異なる）
- **前足部ヒール靴**（forefoot off-loading shoe）を 4–6 週装着するケース多い

## 主な観察項目

- 創部（内側縦切開 or MIS の小切開）
- 母趾の循環・感覚（深腓骨神経領域）
- 腫脹（中足骨骨切り術後は遷延しやすい）
- 装具・テーピングの位置（外反位への戻り防止）

## 退院指導

- 入浴・シャワー（抜糸後）
- 前足部ヒール靴の正しい使用
- 内反テーピングの自己管理
- 復職目安（デスクワーク 2–4 週、立ち仕事 6–8 週）
