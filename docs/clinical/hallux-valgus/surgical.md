# 外反母趾：手術治療

!!! info "骨子（要拡張）"

## 1. 適応

- 保存療法無効の疼痛
- ADL/QOL 障害
- 第2趾転移痛・変形の合併
- **整容目的のみは原則非適応**

## 2. 術式選択（重症度ベース）

| 重症度 | 代表術式 | コメント |
|--------|---------|----------|
| 軽度 (IMA <13°) | **Chevron**（distal osteotomy） | シンプル、回復早い |
| 中等度 (IMA 13–20°) | **Scarf, Mau, Ludloff**（diaphyseal） | 矯正力大、安定性良好 |
| 重度 (IMA >20°) | **Lapidus**（第1TMT 固定）、proximal osteotomy | 過可動性合併に強い |
| 関節破壊例 | **第1MTP 関節固定** | 確実な除痛、可動性は失う |
| DMAA 大 | **biplanar chevron**（DMAA 補正） | 関節適合性復元 |

## 3. MIS（minimally invasive surgery）

- **percutaneous chevron, MICA** が普及
- 小皮切で骨切り → スクリュー固定
- 軟部組織損傷少、回復早い可能性
- **学習曲線あり**、術中透視必須

## 4. 標準術式：Chevron 骨切り術（観血）

| ステップ | ポイント |
|---|---|
| 体位 | 仰臥位、ターニケット |
| 皮切 | 内側または背内側、4–5 cm |
| 神経 | **背側内側神経**（皮切背側）に注意 |
| 関節包切開 | L 字または逆 L 字 |
| 内側突出切除 | bunionectomy（過切除注意） |
| 骨切り | 中足骨頭部に V 字（60°） |
| 整復 | 中足骨頭を外側転位 |
| 固定 | スクリュー 1–2 本（headless or compression） |
| 軟部処置 | 内側関節包縫合、外側 release（必要時） |
| 閉創 | 真皮 + 表皮、ドレーン不要 |

## 5. 合併症

| 合併症 | 頻度 | 対策 |
|--------|------|------|
| 偽関節 | 1–5% | 適切な骨接合、禁煙 |
| 過矯正（hallux varus） | 2–5% | 軟部処置の精緻化 |
| 過小矯正・再発 | 5–15% | 適切な術式選択（重症度ベース） |
| 第1MTP 関節拘縮 | 5–10% | 早期ROM 訓練 |
| 転移性中足骨頭痛 | 5–10% | 骨切り時の dorsiflexion 回避 |
| 神経損傷（背側内側神経） | 1–3% | 皮切設計 |
| 感染 | <1% | 標準的予防 |
