# 理学療法士向け：外反母趾

!!! info "本ページは準備中（骨子のみ）"

## 術後リハ概要

| 時期 | 荷重 | 内容 |
|------|------|------|
| 0–2 週 | 踵荷重（前足部ヒール靴） | 浮腫管理、足趾自動運動 |
| 2–6 週 | 踵荷重継続、徐々に拇趾も | MTP 受動的可動域、内在筋訓練 |
| 6–12 週 | 通常靴へ移行 | 抵抗運動、バランス、歩行再学習 |
| 3 か月〜 | フリー | スポーツ復帰（術式・骨癒合に応じて） |

## 重点項目

- **拇趾 MTP の屈曲・伸展可動域** 確保（拘縮予防）
- **内在筋トレーニング**（short foot exercise, toe spread）
- **歩行再学習**（拇趾蹴り出しの再獲得）
- **後足部・体幹** の連動評価
