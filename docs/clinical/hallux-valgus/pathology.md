# 外反母趾：病態

!!! info "骨子（要拡張）"

## 1. 概念

- 母趾が外側へ偏位、第1中足骨が内反する変形
- 進行で内側突出（バニオン）、種子骨亜脱臼、第1MTP 関節適合性破綻
- 二次的に第2趾ハンマー趾、転移性中足骨頭痛

## 2. 疫学

- 成人女性 **23%**、高齢者 **35%**（メタアナリシス）
- 男女比 1:9 程度

## 3. リスク因子

- 女性
- 家族歴（強い遺伝因子）
- 不適切な靴（ハイヒール、先細）
- **扁平足、第1TMT 関節過可動性**
- 関節弛緩（Marfan, Ehlers-Danlos）
- 関節リウマチ

## 4. 進行のメカニズム

1. 第1中足骨内反 → 母趾の外側付着筋優位
2. 母趾外反進行 → 内側関節包伸展、外側関節包短縮
3. 種子骨が外側偏位 → 内在筋作用の偏位
4. 第1MTP 関節適合性破綻 → 不可逆な変形
