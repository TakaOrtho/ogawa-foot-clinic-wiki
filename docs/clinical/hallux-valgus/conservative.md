# 外反母趾：保存治療

!!! info "骨子（要拡張）"

## 1. 限界の認識

- 保存療法は **症状緩和** には有効
- ただし **変形進行を抑制するエビデンスは限定的**
- 患者に「治す」のではなく「うまく付き合う」治療と説明

## 2. 靴指導

- **Toe box が広い** 靴
- ヒール 3 cm 以下
- 革質はやわらかいもの（バニオン部の圧迫回避）
- 整形靴・オーダーメイド靴の検討

## 3. 装具

- **Toe spacer**（第1-2趾間）
- 夜間スプリント
- **足底板**（内側縦アーチサポート、metatarsal pad）
- バニオン保護パッド

## 4. 運動療法

- **内在筋トレーニング**（short foot exercise, toe spread）
- タオルギャザー、足趾じゃんけん
- 母趾外転筋の強化
- アキレス腱・腓腹筋ストレッチ（前足部負荷軽減）

## 5. 薬物療法

- NSAIDs（炎症期）
- 局所注射は限定的（OA 合併時に検討）

## 6. 手術へ進む判断

- 保存療法 **6 か月以上** で疼痛・ADL 障害が続く
- 整容のみの理由は原則 **手術非適応**
- 患者の年齢・活動性・期待値を踏まえた共同意思決定
