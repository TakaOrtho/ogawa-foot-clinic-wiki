# 外反母趾（Hallux Valgus）

!!! info "本ページは準備中（骨子のみ）"

## 1. 概要

- 母趾が外側へ偏位、第1中足骨が内反する変形
- 有病率: 成人女性 23%、高齢者 35%（メタアナリシス）
- リスク: 女性、家族歴、ハイヒール、扁平足、関節弛緩

## 2. 病態

- 第1中足骨内反、母趾外反、内側突出（バニオン）
- 種子骨亜脱臼、第1MTP 関節適合性破綻
- 進行で第2趾ハンマー趾、転移性中足骨頭痛

## 3. 評価

### 3-1. 画像（荷重位）

| 指標 | 正常 | 軽度 | 中等度 | 重度 |
|------|------|------|--------|------|
| **HVA**（外反母趾角） | <15° | 15–25° | 25–40° | >40° |
| **IMA**（中足骨間角） | <9° | 9–13° | 13–20° | >20° |
| **DMAA** | <10° | — | — | — |

### 3-2. 評価項目

- 第1MTP 関節適合性
- 関節可動性（hypermobility）
- 第1TMT 関節不安定性
- 後足部アライメント

## 4. 治療

### 4-1. 保存療法

- 靴指導（toe box の広い靴）
- 装具（toe spacer, splint）
- 内在筋トレーニング
- 鎮痛
- *進行抑制エビデンスは限定的*

### 4-2. 手術適応

- 保存療法無効の疼痛
- ADL/QOL 障害
- 整容目的のみは原則非適応

### 4-3. 主な術式選択

| 重症度 | 術式例 |
|--------|--------|
| 軽度 | Chevron（distal metatarsal osteotomy） |
| 中等度 | Scarf, Mau, Ludloff（diaphyseal osteotomy） |
| 重度 | Lapidus（第1TMT 関節固定）, proximal osteotomy |
| 関節破壊例 | 第1MTP 関節固定術 |
| 関節適合性破綻 | DMAA 補正（biplanar chevron） |

### 4-4. 最近の動向

- **MIS（minimally invasive surgery）** ─ percutaneous chevron, MICA
- 関節固定の積極適用（hypermobility, 重度）

## 5. 合併症

- 偽関節、過矯正（hallux varus）、過小矯正・再発
- 第1MTP 関節拘縮、転移性中足骨頭痛
- 感染、神経損傷
