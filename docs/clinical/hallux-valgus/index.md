# 外反母趾（Hallux Valgus）

!!! abstract "要点"
    - 手術適応の判断軸は **「変形の大きさ」ではなく「第2・3 中足骨頭痛 (metatarsalgia) などの疼痛・ADL障害」**
    - 当院標準術式は **MICA（Minimally Invasive Chevron-Akin）** — 経皮的に **低速回転バー** で骨切り、スクリュー固定
    - 第2・3 中足骨頭痛合併例には **DMMO（中足骨頸部骨切り、固定なし）** を同時併施
    - 後療法は MICA・DMMO 共通：**弾性包帯圧迫 → 専用サンダル6週 → スニーカー**、**翌日からMTP-ROM**（拘縮予防が最重要）
