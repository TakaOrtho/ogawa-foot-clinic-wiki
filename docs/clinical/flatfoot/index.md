# 扁平足（成人期扁平足障害, Adult Acquired Flatfoot Deformity: AAFD）

!!! abstract "要点"
    - 扁平足は **柔軟性（Flexible）** と **強直性（Rigid）** の2タイプに大別される
    - **Flexible**（早期〜中期）：受動的にアーチが回復する → **靱帯再建**（spring ligament 再建 + FDL 腱移行 + 内側スライディング踵骨骨切り MDCO が標準）
    - **Rigid**（進行期）：拘縮し受動的にも矯正不能 → **骨切り（後足部骨切り）または三関節固定**
    - **共通の後療法**：**6週間非荷重** + **3か月サポーター着用**
    - 主因は **後脛骨筋腱機能不全（PTTD）**。糖尿病・肥満・関節リウマチがリスク
