# 外側側副靱帯損傷:病態・診断

> 本ページは **病態** と **診断** を統合しています。

!!! info "骨子（要拡張）"

## 1. 疫学

- 足関節捻挫の **約 85%** が外側側副靱帯（ATFL ± CFL）の損傷
- 年間発生率: 一般人口で **2.15/1,000 人年**
- スポーツ外傷の中で最も頻度が高い損傷の一つ

## 2. 解剖

- **前距腓靱帯（ATFL）** — 最も損傷頻度が高い
- **踵腓靱帯（CFL）** — 中等度〜重度損傷で併発
- **後距腓靱帯（PTFL）** — 単独損傷はまれ

## 3. 受傷機転

- 底屈・内反位での体重負荷（jump landing, mis-step）
- 不整地走行
- スポーツでのコンタクト・接触

## 4. 重症度分類

| Grade | 病態 | 所見 |
|-------|------|------|
| I | 微細損傷 | 軽度腫脹、荷重可能 |
| II | 部分断裂 | 中等度腫脹・皮下出血、荷重困難 |
| III | 完全断裂 | 著明腫脹、関節不安定、荷重不能 |

## 5. 自然経過

- 適切な治療で多くは回復
- **30–40%** に慢性化症状（不安定感、再受傷、痛み）→ [足関節不安定症](../ankle-instability/pathology.md)

## 1. 病歴

- 受傷機転（内反捻挫が典型）
- 受傷時の音（pop 音は重度示唆）
- 直後の荷重可否
- 既往の捻挫歴

## 2. 身体所見

- **圧痛部位**: ATFL（外果前下方）、CFL（外果遠位）、第5中足骨基部、腓骨筋腱、内側
- 腫脹・皮下出血の範囲
- 急性期の機能テストは制限あり → **受傷後 4–5 日に再評価** が推奨

## 3. Ottawa Ankle Rules（X 線撮像の適応）

以下のいずれかで X 線撮像を考慮:

- 外果後縁または先端の圧痛
- 内果後縁または先端の圧痛
- 第5中足骨基部の圧痛
- 舟状骨の圧痛
- 受傷直後または救急で **4歩以上歩行不能**

## 4. 画像検査

| 検査 | 目的 |
|------|------|
| 単純X線 | 骨折除外（Ottawa rules 適応時） |
| MRI | 重症例、合併損傷疑い、保存療法不応例 |
| エコー | 急性期診断、動的評価 |
| ストレスX線 | 慢性期の不安定性評価（急性期では疼痛で不可） |

## 5. 鑑別・合併損傷

- 第5中足骨基部骨折（Jones / avulsion）
- 距骨骨軟骨損傷（OLT）
- 腓骨筋腱損傷
- **高位足関節捻挫**（syndesmosis injury）
- 内側靱帯損傷（deltoid）
