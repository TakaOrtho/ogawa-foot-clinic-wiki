# 理学療法士向け：外側側副靱帯損傷

!!! info "本ページは準備中（骨子のみ）"

## 急性期リハビリ（Grade I-II）

| 時期 | 目標 |
|------|------|
| 0–3日 | 浮腫・疼痛コントロール、PROM 開始 |
| 3–7日 | 装具下で部分荷重、AROM、等尺収縮 |
| 1–2週 | 全荷重、抵抗運動、固有感覚訓練 |
| 2–6週 | 機能訓練、スポーツ特異訓練 |
| 6週〜 | 競技復帰、ブレース併用 |

## 評価

- CAIT スコア
- Hop test, Y-balance
- 健側比 ≥ 90% で復帰
