# 外側側副靱帯損傷：診断

!!! info "骨子（要拡張）"

## 1. 病歴

- 受傷機転（内反捻挫が典型）
- 受傷時の音（pop 音は重度示唆）
- 直後の荷重可否
- 既往の捻挫歴

## 2. 身体所見

- **圧痛部位**: ATFL（外果前下方）、CFL（外果遠位）、第5中足骨基部、腓骨筋腱、内側
- 腫脹・皮下出血の範囲
- 急性期の機能テストは制限あり → **受傷後 4–5 日に再評価** が推奨

## 3. Ottawa Ankle Rules（X 線撮像の適応）

以下のいずれかで X 線撮像を考慮:

- 外果後縁または先端の圧痛
- 内果後縁または先端の圧痛
- 第5中足骨基部の圧痛
- 舟状骨の圧痛
- 受傷直後または救急で **4歩以上歩行不能**

## 4. 画像検査

| 検査 | 目的 |
|------|------|
| 単純X線 | 骨折除外（Ottawa rules 適応時） |
| MRI | 重症例、合併損傷疑い、保存療法不応例 |
| エコー | 急性期診断、動的評価 |
| ストレスX線 | 慢性期の不安定性評価（急性期では疼痛で不可） |

## 5. 鑑別・合併損傷

- 第5中足骨基部骨折（Jones / avulsion）
- 距骨骨軟骨損傷（OLT）
- 腓骨筋腱損傷
- **高位足関節捻挫**（syndesmosis injury）
- 内側靱帯損傷（deltoid）
