# 外側側副靱帯損傷（Lateral Ankle Ligament Injury）

!!! info "本ページは準備中（骨子のみ）"
    急性期捻挫を中心に、慢性化（足関節不安定症）との連続性を扱います。詳細は順次拡充します。

## 1. 概要

足関節捻挫の **約 85%** が外側側副靱帯（ATFL ± CFL）の損傷。年間発生率 2.15/1,000 人年。

## 2. 解剖

- **前距腓靱帯（ATFL）** — 最も損傷頻度が高い
- **踵腓靱帯（CFL）** — 中等度〜重度損傷で併発
- **後距腓靱帯（PTFL）** — 単独損傷はまれ

## 3. 重症度分類

| Grade | 病態 | 所見 |
|-------|------|------|
| I | 微細損傷 | 軽度腫脹、荷重可能 |
| II | 部分断裂 | 中等度腫脹・皮下出血、荷重困難 |
| III | 完全断裂 | 著明腫脹、関節不安定、荷重不能 |

## 4. 診断

- **Ottawa Ankle Rules** で X 線適応を判断
- 急性期診断: 圧痛部位、前方引き出しテスト（5日後再評価が推奨）
- MRI/エコー: 重症例、合併損傷疑い

## 5. 治療

### 5-1. 保存治療（第一選択）

- **PRICE → POLICE**（Protection, Optimal Loading, Ice, Compression, Elevation）
- 短期 **機能的装具**（lace-up brace, semirigid brace）
- 段階的荷重・運動再開
- 早期可動域訓練が再受傷予防に重要

### 5-2. 手術適応

- 急性期では限定的（重症 III 度＋若年アスリート、靱帯断端の引き込み等）
- **慢性不安定症** へ移行した場合 → [足関節不安定症の手術](../ankle-instability/index.md) 参照

## 6. 注意すべき合併損傷

- 第5中足骨基部骨折
- 距骨骨軟骨損傷（OLT）
- 腓骨筋腱損傷
- 高位足関節捻挫（syndesmosis injury）
- 内側靱帯損傷（deltoid）
