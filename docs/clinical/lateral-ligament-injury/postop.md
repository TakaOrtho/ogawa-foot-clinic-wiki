# 外側側副靱帯損傷：後療法

!!! info "骨子（要拡張）"

## 1. 急性期 → 復帰までの後療法

[保存治療](conservative.md) の段階的リハビリ参照。

### 重要観察項目

- 腫脹・皮下出血の経過
- 荷重時痛（VAS）
- ROM の左右差
- 再受傷不安（CAIT で評価）

## 2. 手術が行われた場合

→ [足関節不安定症の後療法](../ankle-instability/postop.md) を参照

- フェーズ別プロトコル（Phase 1–4）
- 機能評価バッテリー
- 危険サインと退院指導

## 3. 慢性化予防

- 再受傷予防プログラム（神経筋制御、固有感覚、バランス）
- **スポーツ復帰時のブレース** を 6–12 か月
- ホームエクササイズの継続

## 4. 復帰目安

| 重症度 | スポーツ復帰目安 |
|--------|------------------|
| Grade I | 1–3 週 |
| Grade II | 3–6 週 |
| Grade III | 6–12 週（手術なし）／3–6 か月（手術後） |
