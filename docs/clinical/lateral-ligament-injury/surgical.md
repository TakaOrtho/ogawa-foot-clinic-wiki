# 外側側副靱帯損傷:手術治療と後療法

> 本ページは **手術治療** と **後療法(周術期管理+リハビリ)** を統合しています。

!!! info "骨子（要拡張）"

## 1. 急性期の手術適応（限定的）

外側側副靱帯損傷の **急性期手術は原則として行わない**。理由:

- 保存療法で大多数が良好に治癒
- 早期手術と保存療法のアウトカムが同等とのエビデンス

### 急性期手術を検討する例外

- Grade III かつ若年・高活動アスリート（早期復帰要件）
- 靱帯断端の関節内引き込み（impingement 原因）
- 大きな骨片を伴う剥離骨折
- 開放性損傷・他の手術が必要な複合損傷

## 2. 慢性化した場合

→ [足関節不安定症の手術治療](../ankle-instability/surgical.md) を参照

- Modified Broström-Gould 法
- 関節鏡視下 Broström 法
- 解剖学的再建術

## 3. 合併損傷への対応

| 合併損傷 | 対応 |
|----------|------|
| 第5中足骨基部骨折（Jones） | 競技志向で内固定検討 |
| 距骨骨軟骨損傷 | 鏡視下デブリードマン or 骨軟骨処置 |
| 腓骨筋腱断裂 | 修復・腱移行 |
| 高位足関節捻挫 | screw / suture button 固定 |

## 1. 急性期 → 復帰までの後療法

[保存治療](conservative.md) の段階的リハビリ参照。

### 重要観察項目

- 腫脹・皮下出血の経過
- 荷重時痛（VAS）
- ROM の左右差
- 再受傷不安（CAIT で評価）

## 2. 手術が行われた場合

→ [足関節不安定症の後療法](../ankle-instability/surgical.md) を参照

- フェーズ別プロトコル（Phase 1–4）
- 機能評価バッテリー
- 危険サインと退院指導

## 3. 慢性化予防

- 再受傷予防プログラム（神経筋制御、固有感覚、バランス）
- **スポーツ復帰時のブレース** を 6–12 か月
- ホームエクササイズの継続

## 4. 復帰目安

| 重症度 | スポーツ復帰目安 |
|--------|------------------|
| Grade I | 1–3 週 |
| Grade II | 3–6 週 |
| Grade III | 6–12 週（手術なし）／3–6 か月（手術後） |
