# 変形性足関節症：保存治療

!!! info "骨子（要拡張）"

## 1. 体重・活動修正

- BMI 25 以上は減量を強く推奨
- 衝撃の大きい活動の制限（ジョギング、ジャンプ）
- 低衝撃運動（水泳、自転車、エリプティカル）の推奨

## 2. 装具・足底板

- **AFO（足関節装具）** — 進行例で疼痛軽減・歩行改善
- 足底板（後足部矯正、衝撃吸収）
- ロッカーソール靴（前後ロッカー）
- 軟性 ankle brace（軽症〜中等症）

## 3. 薬物療法

- **アセトアミノフェン** が第一選択
- NSAIDs（外用・経口、消化器・腎・心血管リスク評価）
- **関節内注射**:
    - ヒアルロン酸（HA） — 短期効果
    - ステロイド — 急性増悪期、効果数か月

## 4. 理学療法

- 可動域維持（特に背屈拘縮の予防）
- 筋力強化（下腿三頭筋、前脛骨筋、内在筋）
- 歩容修正、装具併用評価
- バランス・固有感覚（転倒予防）

## 5. 補助療法

- 体外衝撃波（限定的エビデンス）
- PRP / BMC（保険外、エビデンス確立途上）

## 6. 手術へ進む判断

- 適切な保存療法 6–12 か月で疼痛・ADL 障害が続く
- 進行性軸ずれ
- 患者の **年齢・活動度・併存疾患** を踏まえた共同意思決定
- 手術選択の患者教育（[手術治療](surgical.md) 参照）
