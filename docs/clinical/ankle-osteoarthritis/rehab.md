# 理学療法士向け：変形性足関節症

!!! info "本ページは準備中（骨子のみ）"

## 保存期リハ

- 体重管理サポート
- 可動域維持（特に背屈）
- 筋力（下腿三頭筋、前脛骨筋、内在筋）
- 歩容修正、装具併用評価
- 衝撃軽減（クッション靴、足底板）

## 術後リハ（術式別）

### 遠位脛骨骨切り術

| 時期 | 内容 |
|------|------|
| 0–6 週 | 非荷重、ROM 維持、近位筋力維持 |
| 6–12 週 | 部分荷重 → 全荷重、ROM 漸増 |
| 3–6 か月 | 機能訓練、歩容再学習 |

### 足関節固定術

- 関節可動性は失われる → **代償運動（距骨下・足根間）** の活用が中心
- 歩容再学習、靴・足底板調整

### 人工足関節置換術

| 時期 | 内容 |
|------|------|
| 0–4 週 | 装具内荷重、創部優先、軽い ROM |
| 4–12 週 | フリーROM、筋力、歩容 |
| 3–6 か月 | 低衝撃スポーツ復帰（ハイインパクトは原則非推奨） |
