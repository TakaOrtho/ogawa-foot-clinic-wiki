# 変形性足関節症：病態

!!! info "骨子（要拡張）"

## 1. 特徴

- 大関節 OA の中では **股・膝に比べ二次性が多い**
- **post-traumatic OA: 70–80%**（過去の足関節骨折・靱帯損傷）
- 一次性 OA は **10–15%** と少数

## 2. 主因

- 足関節骨折（特に Pilon 骨折、二・三果骨折）
- 反復捻挫 → 慢性足関節不安定症
- 距骨骨軟骨損傷（OLT）の蓄積
- 後足部アライメント異常（内反足、扁平足）
- 関節リウマチ、痛風、血友病性関節症

## 3. 病態の特徴

- 距腿関節中心の軟骨摩耗、骨棘形成
- **内反型が多い**（後足部内反、外側靱帯破綻、距骨内反傾斜）
- 距骨下関節・横足根関節への波及

## 4. 自然経過

- 進行性、ただし速度は個人差大
- 内反変形進行 → 立位不安定 → 隣接関節への二次負担
- 末期: 強い疼痛、歩行困難、就労継続困難
