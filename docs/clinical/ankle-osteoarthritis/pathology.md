# 変形性足関節症:病態・診断

> 本ページは **病態** と **診断** を統合しています。

!!! info "骨子（要拡張）"

## 1. 特徴

- 大関節 OA の中では **股・膝に比べ二次性が多い**
- **post-traumatic OA: 70–80%**（過去の足関節骨折・靱帯損傷）
- 一次性 OA は **10–15%** と少数

## 2. 主因

- 足関節骨折（特に Pilon 骨折、二・三果骨折）
- 反復捻挫 → 慢性足関節不安定症
- 距骨骨軟骨損傷（OLT）の蓄積
- 後足部アライメント異常（内反足、扁平足）
- 関節リウマチ、痛風、血友病性関節症

## 3. 病態の特徴

- 距腿関節中心の軟骨摩耗、骨棘形成
- **内反型が多い**（後足部内反、外側靱帯破綻、距骨内反傾斜）
- 距骨下関節・横足根関節への波及

## 4. 自然経過

- 進行性、ただし速度は個人差大
- 内反変形進行 → 立位不安定 → 隣接関節への二次負担
- 末期: 強い疼痛、歩行困難、就労継続困難

## 1. 病歴

- 過去の足関節外傷（骨折・捻挫の詳細・時期）
- 疼痛部位（前内側 / 前外側 / 後方）
- 歩き始め痛、活動後痛、夜間痛
- 可動域制限の認識
- 仕事・ADL への影響

## 2. 身体所見

- 跛行（antalgic gait、内反変形）
- 後足部アライメント（立位・荷重位での観察）
- 関節可動域（背屈・底屈、健側比）
- 圧痛部位、骨棘の触知
- 関節水腫
- 距骨下関節・横足根関節の可動性
- 神経学的所見（腓骨神経）

## 3. 画像検査

| 検査 | 目的 |
|------|------|
| **荷重位X線**（正面・側面・モーティス） | 標準。関節裂隙、骨棘、軸ずれ |
| 後足部アライメントビュー（Saltzman） | 後足部内反/外反角 |
| **Weight-bearing CT（WBCT）** | 3 次元評価、距骨下・後足部の正確な評価 |
| MRI | 軟骨・骨髄浮腫、術前精査 |
| ストレスX線 | 残存不安定性の評価 |

## 4. 分類

### Takakura-Tanaka 分類（内反型）

| Stage | 所見 |
|-------|------|
| I | 早期硬化、軟骨保持 |
| II | 内側関節裂隙狭小化 |
| IIIa | 関節裂隙消失（前内側） |
| IIIb | 関節裂隙消失（内側全体） |
| IV | 距骨の傾き、関節破壊 |

### COFAS Classification

- アライメントベース（内反 / 中立 / 外反）+ 隣接関節評価

## 5. 鑑別

- 関節リウマチ・他炎症性関節炎
- 痛風・偽痛風
- 距骨壊死
- Charcot 関節（特に糖尿病性）
- 距骨骨軟骨損傷の単独病変
