# 変形性足関節症：手術治療

!!! info "骨子（要拡張）"

## 1. 適応・術式選択

| 病期・条件 | 第一選択 |
|------------|---------|
| 早期、軸ずれ伴う、軟骨残存 | **遠位脛骨骨切り術（SMOT / DTO）** |
| 早期、軟骨損傷限局 | 関節鏡視下デブリードマン、骨軟骨処置 |
| 進行期、若年・高活動 | **足関節固定術（Ankle Arthrodesis）** |
| 進行期、高齢・低活動、可動性温存希望 | **人工足関節置換術（Total Ankle Arthroplasty, TAA）** |
| 後足部変形合併 | 後足部矯正骨切り術の併施 |

## 2. 遠位脛骨骨切り術（SMOT / Supramalleolar Osteotomy）

- 関節温存術式（joint preservation）
- 内反型 OA に対し開大式（medial opening wedge）が一般的
- **適応**: Takakura II–IIIa、軟骨残存、若年〜中年
- プレート + 骨移植
- 骨癒合 8–12 週

## 3. 足関節固定術（Ankle Arthrodesis）

- **長期エビデンス豊富**、確実な除痛
- アプローチ: 前方、外側、後方、関節鏡視下
- 固定法: スクリュー、プレート、外固定器
- 課題: **隣接関節 OA**（距骨下、横足根、Chopart）の長期進行
- 骨癒合まで **非荷重 6–12 週**

### Arthrodesis のメリット/デメリット

| メリット | デメリット |
|---------|----------|
| 確実な除痛 | 関節可動性喪失 |
| 長期耐久 | 隣接関節 OA 進行 |
| 重労働可能 | 不整地歩行困難 |
| 感染にも対応可能 | 偽関節 5–10% |

## 4. 人工足関節置換術（TAA）

- 関節可動性温存、歩容改善、QOL 向上
- 機種の世代改善で **10 年生存 80–90%**
- 適応拡大中（若年は慎重）
- ハイインパクトスポーツは原則非推奨

### TAA のメリット/デメリット

| メリット | デメリット |
|---------|----------|
| 関節可動性温存 | 長期耐久が固定術より劣る |
| 歩容自然 | 再手術（revision）が複雑 |
| 隣接関節への負担減 | 感染で救肢困難になる例 |
| 早期 QOL 改善 | 機種固有の合併症 |

### TAA 適応の慎重条件

- 60 歳以上、低〜中等活動度
- 大きな軸ずれなし（術中補正可能）
- 良好な骨質
- 距骨壊死なし
- 過去の感染歴なし
- 重度肥満ではない

## 5. 選択の判断軸

```
進行期足関節 OA
  ├─ 若年（<50歳）+ 高活動 + 重労働 → Arthrodesis
  ├─ 中高年（50–70）+ 中活動 → TAA を第一選択（適応評価）
  ├─ 高齢（>70）+ 低活動 + 良好骨質 → TAA
  └─ 距骨壊死 / 感染既往 / 重度軸ずれ → Arthrodesis
```

## 6. 後足部変形合併時の併施手術

| 合併変形 | 対応 |
|----------|------|
| 後足部内反 | 踵骨外反骨切り（Dwyer） |
| 後足部外反 | 踵骨内反骨切り、内側スライディング |
| 横足根 OA | 後足部三関節固定との併施 |
| アキレス腱拘縮 | 経皮腱延長 |

## 7. 合併症

| 合併症 | Arthrodesis | TAA |
|--------|-------------|-----|
| 偽関節 | 5–15%（喫煙・DM で増） | — |
| 隣接関節 OA | 長期で増加 | 比較的少 |
| インプラント loosening | — | 5–10%（10年） |
| 感染（深部） | 1–3% | 1–3%（救肢困難） |
| 神経・血管損傷 | <5% | <5% |
| 創部問題 | 5–10% | 5–15% |
