# 変形性足関節症:手術治療と後療法

> 本ページは **手術治療** と **後療法(周術期管理+リハビリ)** を統合しています。

!!! info "骨子（要拡張）"

## 1. 適応・術式選択

| 病期・条件 | 第一選択 |
|------------|---------|
| 早期、軸ずれ伴う、軟骨残存 | **遠位脛骨骨切り術（SMOT / DTO）** |
| 早期、軟骨損傷限局 | 関節鏡視下デブリードマン、骨軟骨処置 |
| 進行期、若年・高活動 | **足関節固定術（Ankle Arthrodesis）** |
| 進行期、高齢・低活動、可動性温存希望 | **人工足関節置換術（Total Ankle Arthroplasty, TAA）** |
| 後足部変形合併 | 後足部矯正骨切り術の併施 |

## 2. 遠位脛骨骨切り術（SMOT / Supramalleolar Osteotomy）

- 関節温存術式（joint preservation）
- 内反型 OA に対し開大式（medial opening wedge）が一般的
- **適応**: Takakura II–IIIa、軟骨残存、若年〜中年
- プレート + 骨移植
- 骨癒合 8–12 週

## 3. 足関節固定術（Ankle Arthrodesis）

- **長期エビデンス豊富**、確実な除痛
- アプローチ: 前方、外側、後方、関節鏡視下
- 固定法: スクリュー、プレート、外固定器
- 課題: **隣接関節 OA**（距骨下、横足根、Chopart）の長期進行
- 骨癒合まで **非荷重 6–12 週**

### Arthrodesis のメリット/デメリット

| メリット | デメリット |
|---------|----------|
| 確実な除痛 | 関節可動性喪失 |
| 長期耐久 | 隣接関節 OA 進行 |
| 重労働可能 | 不整地歩行困難 |
| 感染にも対応可能 | 偽関節 5–10% |

## 4. 人工足関節置換術（TAA）

- 関節可動性温存、歩容改善、QOL 向上
- 機種の世代改善で **10 年生存 80–90%**
- 適応拡大中（若年は慎重）
- ハイインパクトスポーツは原則非推奨

### TAA のメリット/デメリット

| メリット | デメリット |
|---------|----------|
| 関節可動性温存 | 長期耐久が固定術より劣る |
| 歩容自然 | 再手術（revision）が複雑 |
| 隣接関節への負担減 | 感染で救肢困難になる例 |
| 早期 QOL 改善 | 機種固有の合併症 |

### TAA 適応の慎重条件

- 60 歳以上、低〜中等活動度
- 大きな軸ずれなし（術中補正可能）
- 良好な骨質
- 距骨壊死なし
- 過去の感染歴なし
- 重度肥満ではない

## 5. 選択の判断軸

```
進行期足関節 OA
  ├─ 若年（<50歳）+ 高活動 + 重労働 → Arthrodesis
  ├─ 中高年（50–70）+ 中活動 → TAA を第一選択（適応評価）
  ├─ 高齢（>70）+ 低活動 + 良好骨質 → TAA
  └─ 距骨壊死 / 感染既往 / 重度軸ずれ → Arthrodesis
```

## 6. 後足部変形合併時の併施手術

| 合併変形 | 対応 |
|----------|------|
| 後足部内反 | 踵骨外反骨切り（Dwyer） |
| 後足部外反 | 踵骨内反骨切り、内側スライディング |
| 横足根 OA | 後足部三関節固定との併施 |
| アキレス腱拘縮 | 経皮腱延長 |

## 7. 合併症

| 合併症 | Arthrodesis | TAA |
|--------|-------------|-----|
| 偽関節 | 5–15%（喫煙・DM で増） | — |
| 隣接関節 OA | 長期で増加 | 比較的少 |
| インプラント loosening | — | 5–10%（10年） |
| 感染（深部） | 1–3% | 1–3%（救肢困難） |
| 神経・血管損傷 | <5% | <5% |
| 創部問題 | 5–10% | 5–15% |

## 1. 共通の周術期注意点

- 高齢患者が多い → 認知・転倒・せん妄リスク
- **抗血栓薬の管理**（術後 DVT 予防 vs 出血リスク）
- 栄養・皮膚（圧迫潰瘍リスク）
- **禁煙指導厳格に**（偽関節・感染リスク）
- 糖尿病・末梢循環障害は感染リスク高

## 2. 術式別プロトコル

### 2-1. 遠位脛骨骨切り術（SMOT）

| 時期 | 荷重 | リハ |
|------|------|------|
| 0–6 週 | **非荷重** | ROM 維持、近位筋力維持 |
| 6–8 週 | 部分荷重 | ROM 漸増、軽筋力 |
| 8–12 週 | 全荷重 | 筋力、バランス |
| 3–6 か月 | フリー | 歩容再学習、活動復帰 |

### 2-2. 足関節固定術

| 時期 | 荷重 | リハ |
|------|------|------|
| 0–2 週 | 非荷重、シーネ | 浮腫管理、足趾運動 |
| 2–6 週 | 非荷重、ブーツ | ROM は距骨下・横足根中心 |
| 6–12 週 | 段階的荷重（骨癒合確認後） | 距骨下・横足根の代償運動活用、歩容再学習 |
| 3–6 か月 | 全荷重・フリー | 靴調整、ロッカーソール、職場復帰 |

!!! tip "固定術後のリハ重点"
    - 距腿関節の可動性は失われる → **距骨下・横足根関節の柔軟性が代償** の鍵
    - **歩容再学習**（ロッカーソール靴の活用）
    - 隣接関節 OA 予防のための継続フォロー

### 2-3. 人工足関節置換術（TAA）

| 時期 | 荷重 | リハ |
|------|------|------|
| 0–2 週 | 装具内 部分荷重（施設プロトコル） | 創部優先、軽 ROM |
| 2–4 週 | 装具内 全荷重 | 自動 ROM 開始 |
| 4–12 週 | 装具 → 通常靴 | 抵抗運動、バランス、歩容 |
| 3–6 か月 | フリー | 低衝撃スポーツ復帰可、ハイインパクトは原則非推奨 |

!!! warning "TAA で注意すべき合併症"
    - **感染**（インプラント感染は救肢困難）
    - インプラント周囲骨折・loosening
    - 創傷治癒遅延（前方アプローチで皮膚壊死リスク）

## 3. 観察項目（術式共通）

| 項目 | チェック |
|------|----------|
| 創部 | 滲出、発赤、腫脹、悪臭 |
| 神経 | 腓骨神経麻痺（母趾背屈、第1-2趾間感覚） |
| 循環 | 足趾色調、CRT、足背・後脛骨動脈 |
| 浮腫 | 患肢挙上の徹底 |
| 痛み | NRS、薬効 |
| DVT | ふくらはぎ腫脹、ホーマンズ徴候 |
| 発熱 | 38°C 以上は深部感染要警戒 |

## 4. 退院指導

- 創部・装具の自己管理
- 体重コントロール（OA 進行・術後合併症予防）
- **禁煙継続**（特に固定術）
- 通院・通所リハ調整
- 危険サインの教育
- 復職・スポーツ復帰の現実的期待値

## 5. 復職目安

| 仕事 | 固定術 | TAA |
|------|--------|-----|
| デスクワーク | 6–8 週 | 4–6 週 |
| 立ち仕事 | 12 週〜 | 8–12 週 |
| 重労働 | 6 か月〜（推奨されないことも） | 推奨されない |
| ランニング | 不可〜慎重 | 不可〜慎重 |
| 自転車・水泳 | 3–6 か月 | 3 か月 |

## 6. 長期フォロー

- TAA: 年 1 回 X 線でインプラント評価
- 固定術: 隣接関節 OA の経過観察
- 体重・活動レベル管理
