# 変形性足関節症（Ankle Osteoarthritis）

!!! abstract "要点"
    - 股・膝 OA と異なり **二次性が多い**（過去の足関節骨折・反復捻挫が主因 = post-traumatic OA 70–80%）
    - 内反型が多く、後足部アライメント評価（**荷重位 X 線、WBCT**）が術前計画に必須
    - 手術選択軸：**重症度・ADL・年齢・後療法（活動レベル）の希望**
    - **当院の推奨順：骨切り術 → 人工足関節（TAA）→ 関節固定術**
        - **骨切り術**：自分の関節を温存できる **唯一の手術**。時間経過で症状進行の可能性あり
        - **TAA**：可動性温存。**重労働・重量物保持には注意**
        - **関節固定術**：**最終手段だが確実な効果**
