# 変形性足関節症（Ankle Osteoarthritis）

!!! info "本ページは準備中（骨子のみ）"

## 1. 概要

- 大関節 OA の中では **股・膝に比べ二次性が多い**
- 主因: 過去の足関節骨折・靱帯損傷（**post-traumatic OA: 70–80%**）
- 一次性 OA は少数（10–15%）

## 2. 病態

- 距腿関節の軟骨摩耗、骨棘、内反/外反変形を伴うことが多い
- **内反型** が多い（後足部内反、外側靱帯破綻、距骨内反傾斜）

## 3. 評価

### 3-1. 画像

- 単純X線（**荷重位**：正面・側面・モーティス・後足部アライメント）
- **Weight-bearing CT** ─ 3次元評価、後足部・距骨下関節の正確な評価
- MRI ─ 軟骨・骨髄浮腫

### 3-2. 分類（Takakura/Tanaka）

| Stage | 所見 |
|-------|------|
| I | 早期硬化、軟骨保持 |
| II | 内側関節裂隙狭小化 |
| IIIa | 関節裂隙消失（前内側） |
| IIIb | 関節裂隙消失（内側全体） |
| IV | 距骨の傾き、関節破壊 |

## 4. 治療

### 4-1. 保存療法

- 体重管理、活動修正
- 鎮痛（アセトアミノフェン、NSAIDs）
- **装具療法**（足底板、ankle foot orthosis）
- 関節内注射（ヒアルロン酸、ステロイド）
- 理学療法（可動域、筋力、歩行訓練）

### 4-2. 手術適応・術式選択

| 病期・条件 | 第一選択 |
|------------|---------|
| 早期、軸ずれ伴う | **遠位脛骨骨切り術（SMOT/distal tibial osteotomy）** |
| 早期、軟骨損傷限局 | 関節鏡視下デブリードマン、骨軟骨処置 |
| 進行期、若年・高活動 | **足関節固定術（Ankle Arthrodesis）** |
| 進行期、高齢・低活動 | **人工足関節置換術（Total Ankle Arthroplasty, TAA）** |
| 後足部変形合併 | 後足部矯正骨切り術の併施 |

### 4-3. Ankle Arthrodesis vs TAA

- **Arthrodesis**: 確実な除痛、長期耐久、隣接関節 OA が長期問題
- **TAA**: 関節可動性温存、QOL 向上、機種改善で 10 年生存 80–90%
- 選択は **年齢・活動度・骨質・隣接関節状態** で判断

## 5. 合併症

- 偽関節（特に喫煙・糖尿病）
- 隣接関節 OA（arthrodesis 後）
- インプラント周囲骨折・loosening（TAA）
- 感染（深部感染は救肢困難になる例も）
