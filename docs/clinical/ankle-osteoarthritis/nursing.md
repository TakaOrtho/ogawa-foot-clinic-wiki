# 看護師向け：変形性足関節症

!!! info "本ページは準備中（骨子のみ）"

## 術式別の周術期注意点

### 1. 遠位脛骨骨切り術（SMOT）

- プレート固定、骨癒合に **8–12 週**
- 非荷重 4–6 週 → 段階的荷重

### 2. 足関節固定術

- **骨癒合まで非荷重 6–12 週**
- 喫煙・糖尿病で偽関節リスク高 → **禁煙指導厳格に**
- 隣接関節への負担増を退院前に説明

### 3. 人工足関節置換術（TAA）

- 早期荷重可（術式・施設プロトコルに応じて術後 2–4 週）
- **感染予防が最重要**（インプラント感染は救肢困難）
- 創部・滲出・発熱を厳格に観察

## 共通の観察項目

- 高齢患者が多い → 認知・転倒・せん妄リスク
- 抗血栓薬の管理（術後 DVT 予防 vs 出血リスク）
- 栄養・皮膚（圧迫潰瘍リスク）
- 退院後の通院・通所リハ調整
