# 足関節不安定症：後療法（周術期管理・術後リハビリテーション）

!!! abstract "押さえるポイント"
    - 術前: **皮膚・末梢循環・喫煙・内服** の最終確認
    - 術直後: **腓骨神経麻痺**（母趾背屈・第1-2趾間感覚）を最優先で確認
    - 病棟: **患肢挙上の徹底**、シーネ内浮腫・圧迫の観察
    - リハ: 内反負荷は **6 週間まで完全禁忌**。機械的安定だけでなく **神経筋制御** の再獲得を重視
    - スポーツ復帰は **時間ではなく機能テスト**（CAIT・hop test・Y-balance）で判定

---

## 1. 術前管理

### 1-1. 入院時アセスメント

| 項目 | 観察・確認 |
|------|-----------|
| 皮膚 | 外果周囲の発赤・水疱・湿疹・白癬・潰瘍。**創部予定部位の異常は手術延期事由** |
| 末梢循環 | 足背動脈・後脛骨動脈触知、足趾の色調・温度、CRT |
| 神経学的所見 | 足趾運動、しびれ・感覚の左右差（既存所見の記録） |
| 既往・合併症 | 糖尿病（HbA1c）、抗血栓薬、ステロイド、関節リウマチ、末梢神経障害 |
| 内服 | 抗血栓薬の休薬指示確認、ステロイド・生物学的製剤の継続可否 |
| 生活習慣 | **喫煙**（創傷治癒・偽関節リスク）、飲酒、栄養状態 |
| 既往の足部外傷 | 反復捻挫の回数、最後の受傷時期、現在の症状程度 |

### 1-2. 患者教育

- 手術の流れ（麻酔、所要時間、術後シーネ、ドレーン原則なし）
- **松葉杖の練習**（術後は非荷重 2 週間）
- 患肢挙上の重要性（心臓より高く、3点支持枕で 24h）
- 術後の疼痛コントロール方法（PCA・経口鎮痛薬）
- 禁飲食時間、内服薬の朝の指示
- 入院中の生活（シャワー、トイレ移動、着替え）

### 1-3. 術前準備（手術当日）

- 抗菌石鹸での下肢洗浄、足趾爪切り、爪垢除去
- マニキュア・ペディキュア除去（パルスオキシメーター用に複数指）
- アクセサリー類除去
- 弾性ストッキング装着（健側のみ、または両側 ※施設プロトコル）
- 抗菌薬投与時刻の確認

---

## 2. 術中・帰室時の引き継ぎ

### 2-1. 手術室からの申し送り

- 術式（観血 / 関節鏡視下、追加術式の有無）
- 出血量、輸血の有無
- ターニケット使用時間
- ドレーン挿入の有無（通常なし、入っていれば部位・形式）
- 術中投与薬（抗菌薬、鎮痛薬、抗血栓薬）
- 関節内処置の有無（OLT処置 → 荷重制限変更の可能性）
- **外固定の種類**（シーネの角度、ブーツの有無）
- 神経ブロックの有無（→ 帰室後の感覚消失部位）

### 2-2. 帰室時バイタル

- 意識、呼吸、循環、SpO₂
- 創部出血量、シーネ内の出血浸潤の範囲（マーキング）
- 疼痛スケール（NRS / VAS）
- **足趾の運動・感覚・色調・冷感**（健側と比較）

---

## 3. 術後病棟管理

!!! danger "緊急で医師報告すべき所見"
    - **足趾の強い痛み・しびれ・冷感・色調変化（蒼白・チアノーゼ）** → コンパートメント症候群、循環障害
    - **シーネ内の急速増大する腫脹**、シーネ圧迫感の訴え
    - **足趾運動消失**（特に母趾背屈不能 → 腓骨神経麻痺）
    - 安静時 NRS ≥ 7、鎮痛薬無効の強い痛み
    - 発熱 ≥ 38.0°C、創部発赤拡大、悪臭ある滲出
    - 呼吸困難・胸痛（→ PE 疑い）

### 3-1. 観察項目（術後 24–48 時間）

| 観察項目 | 頻度（24h） | チェック内容 |
|----------|-------------|--------------|
| バイタル | 4 時間毎 | 体温・血圧・脈・SpO₂ |
| 末梢神経 | 2–4 時間毎 | 母趾背屈、第1-2趾間感覚（深腓骨神経）、足底感覚（脛骨神経）、足背感覚（浅腓骨神経） |
| 末梢循環 | 2–4 時間毎 | 足趾色調、温度、CRT、足背動脈 |
| 疼痛 | 各勤務 | NRS/VAS、鎮痛効果 |
| シーネ・創部 | 各勤務 | 出血量、浸潤、皮膚圧迫所見 |
| 患肢挙上 | 随時 | 心臓より高く、3点支持の維持 |

### 3-2. 神経学的観察の解剖

| 神経 | 評価部位 | 評価方法 |
|------|----------|----------|
| 深腓骨神経 | 第1-2趾間背側 | 触覚、母趾背屈筋力 |
| 浅腓骨神経 | 足背中央〜外側 | 触覚、足部外反筋力 |
| 脛骨神経 | 足底 | 触覚、足趾屈曲 |
| 腓腹神経 | 足部外側縁 | 触覚 |

!!! tip "実践のコツ"
    - **術前の所見を必ず記録**しておく（術後の左右差・術前後差を見るため）
    - 「しびれているか？」と聞くだけでなく、**触って答えてもらう** か、健側と比較させる
    - **腓骨神経麻痺は外側からのシーネ圧迫**でも起こる → シーネ巻き直しを医師と相談

### 3-3. 患肢挙上・浮腫管理

- 心臓よりも高い位置（枕 2–3 個、足挙上クッション）
- **足趾は枕外に出す**（趾の循環観察のため）
- 24 時間体位保持、ベッド上での体位変換時もキープ
- 初回起立・トイレ歩行は **必ず患肢挙上時間を確保した後**

### 3-4. 疼痛管理

- 定時鎮痛（NSAIDs / アセトアミノフェン）+ 頓用
- リハビリ前 30 分に予防的鎮痛
- 24h NRS が安定して 3 以下になることを目標
- 強オピオイドは便秘・悪心予防セットで

### 3-5. DVT 予防

- 術後早期からの **足趾運動（背屈・底屈）** を 1 時間に 10 回
- 健側の積極運動、間欠的空気圧迫
- 抗血栓薬投与時はリスク・出血徴候の観察
- ホーマンズ徴候、ふくらはぎの腫脹・熱感

### 3-6. 外固定（シーネ・ブーツ）管理

- **シーネのずれ・浸出**を毎勤務でチェック
- 患者が「きつい」と訴えたら必ず医師確認（自己判断で巻き直さない）
- 入浴・シャワー時は外固定を濡らさない（防水カバー使用）
- 装具に交換後は装着方法を再指導

## 4. 術後リハビリテーション

### 4-1. リハビリ原則

| 期間 | 組織治癒 | 注意点 |
|------|----------|--------|
| 0–2 週 | 炎症期 | 内反は完全禁忌、患肢挙上、浮腫コントロール |
| 2–6 週 | 増殖期 | 受動的内反は禁止、軽い外反・底背屈はOK |
| 6–12 週 | 再構築期 | 内反負荷を段階的に導入、抵抗運動開始 |
| 3–6 か月 | 成熟期 | スポーツ特異訓練、競技復帰準備 |

### 4-2. リハビリ目標

- 機械的安定性の維持（縫合靱帯への過度負荷を避ける）
- 可動域: 背屈 ≥ 健側 -5°、底屈 健側同等
- 筋力: 健側比 ≥ 90%
- 神経筋制御: Y-balance, SEBT で健側比 ≥ 90%
- 心理的準備（kinesiophobia の軽減）

### 4-3. フェーズ別プロトコル

#### Phase 1：保護期（術後 0–2 週）

**ゴール**: 創傷治癒、浮腫コントロール、患肢適切管理、DVT 予防

| 項目 | 内容 |
|------|------|
| 荷重 | **非荷重**（松葉杖2本） |
| 外固定 | 短下肢シーネ（中間位） |
| 可動域 | シーネ内のため不可。健側・近位関節（膝・股）は自由 |
| 筋力 | 大腿四頭筋・ハムストリングス・殿筋のセッティング |
| 浮腫管理 | 患肢挙上、足趾運動（背屈・底屈）×10/h |
| その他 | 体幹・上肢の機能維持トレーニング |

#### Phase 2：早期可動期（2–6 週）

**ゴール**: シーネ → ブーツ装具へ移行、可動域開始、軽負荷の筋活動

| 項目 | 内容 |
|------|------|
| 荷重 | **部分荷重 → 全荷重**（術後 4 週で装具内全荷重を目標） |
| 外固定 | ブーツ装具（夜間も装着、抜くのはリハビリ・入浴時のみ） |
| 可動域 | **背屈・底屈の自動運動**、ABC文字運動（足趾で空中描画）。**内反は禁忌**、外反・回外は中間位までに留める |
| 筋力 | 腓骨筋・前脛骨筋の **等尺収縮**（中間位）、弾性バンドでの軽い外反・背屈 |
| 浮腫 | リンパドレナージ、間欠的圧迫、ホーザリー |
| 歩行 | 装具・松葉杖併用の段階的荷重歩行 |

!!! warning "禁忌・注意"
    - **強い内反負荷**（distance, surprise stepping, balance board の側方）
    - **長時間の懸垂位（下垂位）** — 縫合部牽引と浮腫増悪
    - **冷感・しびれ訴え時の継続** — 神経学的所見を再評価

#### Phase 3：中期（6–12 週）

**ゴール**: 装具離脱、全可動域回復、筋力 80% 復帰、両脚バランス再獲得

| 項目 | 内容 |
|------|------|
| 荷重 | **完全荷重**、装具なし歩行 |
| 外固定 | 6 週で装具離脱、屋外活動時は **ブレース** で保護 |
| 可動域 | 全方向自動・他動。**内反も慎重に段階的導入** |
| 筋力 | 弾性バンド抵抗運動（外反・内反・背屈・底屈）、カーフレイズ（両脚 → 片脚） |
| バランス | 両脚立位（不安定面）→ 片脚立位（堅固面 → 不安定面） |
| 神経筋 | 固有感覚訓練（目開閉、bilateral → unilateral） |
| 心血管 | 自転車エルゴ、水中歩行（創閉鎖後）、ステアマスター |

#### Phase 4：後期・機能訓練期（3–6 か月）

**ゴール**: スポーツ特異動作の獲得、再受傷予防

| 項目 | 内容 |
|------|------|
| 筋力 | プログレッシブレジスタンストレーニング、片脚 deadlift, lunge |
| バランス | 片脚立位（不安定面）+ 上肢外乱、SEBT, Y-balance |
| プライオメトリクス | bilateral jump → unilateral hop → multidirectional hop |
| アジリティ | ラダー、コーン、change of direction、cutting drill |
| スポーツ特異 | sport-specific drill（競技に応じて） |
| 装具 | スポーツ復帰時は **lace-up または semirigid brace** を 6–12 か月使用推奨 |

## 5. ADL 援助・退院指導

### 5-1. 初回起立・歩行

- 起立性低血圧予防（端座位 → 立位を段階的に）
- 松葉杖 / 歩行器の使用法を再確認
- **非荷重歩行**（術後 2 週）— 患肢を完全に床から離す
- 階段昇降: **上り 健側から**（"Good goes up"）、**下り 患側から**（"Bad goes down"）

### 5-2. ADL 支援

| ADL | 留意点 |
|------|-------|
| トイレ | 患肢挙上できる多目的トイレ、ナースコール位置 |
| シャワー | 防水カバー、シャワーチェア、転倒予防マット |
| 着替え | 患側から着る／健側から脱ぐ（ゆとり優先） |
| 食事 | 患肢挙上したまま摂取できるテーブル配置 |
| 睡眠 | 患肢挙上枕、ナースコール |

### 5-3. 退院前チェックリスト

- [ ] 創部・シーネの自己観察ができる
- [ ] 患肢挙上・松葉杖歩行が自立
- [ ] 鎮痛薬・抗血栓薬の自己管理可
- [ ] **危険サイン** を 5 つ言える（下記）
- [ ] 緊急連絡先を理解
- [ ] 次回外来日程を把握
- [ ] シャワーの可否と方法を理解
- [ ] 復職・スポーツ復帰の目安を理解

### 5-4. 患者・家族に伝える「危険サイン」

!!! warning "すぐに病院に連絡"
    1. 痛みが急に強くなる、薬が効かない
    2. 足趾のしびれ・冷感・色が悪い
    3. シーネ内がきつくて痛い
    4. 創部から膿・悪臭・赤みが広がる
    5. 発熱（38°C以上）が続く
    6. ふくらはぎが腫れて痛い・赤い → DVT
    7. 急な息切れ・胸痛 → 肺塞栓

### 5-5. 日常生活指導

- シャワー: 創部が乾燥・抜糸後（通常 2 週）まで防水カバー
- 入浴: 抜糸後 + 創部完全乾燥後
- 運転: 右足術側は装具除去・自由可動域回復後（おおむね 6–8 週）
- 飲酒・喫煙: **禁煙継続**を必ず伝える
- 食事: 高タンパク、便秘予防に水分・食物繊維

## 6. 機能評価・スポーツ復帰基準

### 6-1. 評価バッテリー

| テスト | 基準 |
|--------|------|
| **CAIT スコア** | ≥ 28/30 |
| **Hop test 4種**（single, triple, crossover, 6m timed） | 健側比 ≥ 90% |
| **Y-balance / SEBT** | 健側比 ≥ 90% |
| **筋力測定**（HHD / isokinetic） | 健側比 ≥ 90%（外反・背屈・底屈） |
| **片脚立位**（目閉、不安定面） | 30 秒以上 |
| **Drop jump / landing 評価** | dynamic valgus, asymmetry なし |

### 6-2. 競技復帰の段階

1. ジョギング（直線）
2. ランニング + change of direction
3. プライオメトリクス
4. 競技特異練習（contact なし）
5. フル練習
6. 試合復帰

各段階で 1–2 週間ずつ、症状なく完了したら次へ。

---

## 7. 外来フォロー

- 退院前: 創部状態、外固定の脱着、松葉杖歩行、シャワーの可否、危険サイン教育
- 外来: 術後 **2 週（抜糸）、6 週（外固定除去）、3 か月、6 か月、1 年**

---

## 8. 進めない・中止すべきサイン

!!! danger "セッション中止・医師相談すべき所見"
    - 著明な腫脹（健側比 1.5 cm 以上の周径差増大）
    - 安静時痛 NRS 5 以上が新規出現
    - 創部発赤・熱感
    - giving way の再出現
    - 神経症状の新規出現（しびれ・筋力低下）

---

## 9. ホームエクササイズ（退院後の最低限セット）

| エクササイズ | 量 |
|--------------|---|
| カーフレイズ（両 → 片脚） | 15 × 3 セット |
| バンド外反 | 15 × 3 セット |
| 片脚立位（目開 30秒 → 目閉） | 1 分 × 3 セット |
| 不安定面片脚立位 | 30 秒 × 3 セット |
