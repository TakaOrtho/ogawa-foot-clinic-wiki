# 足関節不安定症（Chronic Ankle Instability, CAI）

!!! abstract "要点"
    - 足関節捻挫の **20–40%** が慢性化し、慢性足関節不安定症（CAI）に至る
    - 病態は **機械的不安定性**（靱帯の緩み）と **機能的不安定性**（感覚運動制御障害）の混在
    - 初期治療は保存療法（3か月の運動療法）。改善が乏しければ手術を検討
    - 標準術式は **Modified Broström-Gould 法**。近年は **関節鏡視下手術**（arthroscopic Broström）が普及
    - 重度症例・反復術後症例では **解剖学的再建術（autograft/allograft tendon）** や **腓骨筋腱を用いた非解剖的再建** を検討

---

## 1. 解剖と病態

### 1-1. 外側支持機構の解剖

| 靱帯 | 走行 | 役割 |
|------|------|------|
| **前距腓靱帯（ATFL）** | 腓骨遠位前縁 → 距骨外側 | 底屈位での内反制動。最も損傷頻度が高い |
| **踵腓靱帯（CFL）** | 腓骨遠位先端 → 踵骨外側 | 中間位での内反制動。距骨下関節も安定化 |
| **後距腓靱帯（PTFL）** | 腓骨遠位後方 → 距骨後突起 | 背屈位での内反制動。単独損傷はまれ |

### 1-2. 病態分類（Hertel）

- **機械的不安定性（Mechanical Instability）** — 距骨傾斜・前方引き出しの過度
- **機能的不安定性（Functional Instability）** — 固有感覚・神経筋制御の障害（"giving way"）
- 両者は連続体として併存する

### 1-3. 疫学

- 急性足関節捻挫の年間発生率: 一般人口で **2.15/1,000 人年**
- 捻挫後の **30–40%** が慢性化症状を残す
- スポーツ復帰後の **再受傷率: 約 30%**

---

## 2. 診断

### 2-1. 病歴聴取

- 初回受傷機転、回数、最後の受傷時期
- "giving way"（不意の踏み外し感）の頻度
- 機能制限（スポーツ・日常）の評価 → **CAIT（Cumberland Ankle Instability Tool）≤ 24点** で診断的
- 既存スコア: FAAM, FAOS, AOFAS Hindfoot Score

### 2-2. 身体所見

=== "前方引き出しテスト"
    底屈 10–20°、脛骨を後方へ押し付け踵を前方へ引く。
    **健側比 +3 mm 以上、または絶対値 10 mm 以上** で陽性。

=== "内反ストレステスト（Talar Tilt）"
    踵骨を内反させ距骨傾斜角を評価。
    **健側比 5° 以上の差**、または **絶対 10° 以上** で陽性。

=== "Squeeze test / External rotation test"
    高位足関節捻挫（syndesmosis injury）の鑑別。

### 2-3. 画像検査

| 検査 | 目的 |
|------|------|
| 単純X線（正面・側面・モーティス・荷重位） | 骨折除外、変形・OA評価 |
| **ストレスX線**（前方引き出し・内反） | 機械的不安定性の客観化 |
| MRI | 靱帯損傷の質的評価、骨軟骨損傷（OLT）・腓骨筋腱症・距骨下関節病変の検索 |
| 超音波（動的評価） | 外来での即時評価、術後経過追跡 |
| CT（荷重位 / Weight-bearing CT） | 軟部組織以外の3次元評価、後足部アライメント評価 |

### 2-4. 鑑別・合併病変（"Three-bucket" rule）

CAI患者の **約60–80% に合併病変** あり。手術前に必ずスクリーニング:

1. **関節内** — 骨軟骨損傷（OLT）、滑膜炎、遊離体、前外側インピンジメント
2. **関節外（内側）** — 距腿後方インピンジメント、後脛骨筋腱症
3. **関節外（外側）** — 腓骨筋腱断裂・脱臼、腓骨筋下支帯損傷
4. **アライメント** — 後足部内反、凹足、第1中足骨底屈

---

## 3. 保存療法

!!! tip "原則：最低3か月の運動療法"
    急性捻挫後・慢性不安定症ともに **monitored functional rehabilitation** が第一選択。

### プロトコル要素

- **可動域訓練**（背屈・底屈・内外反）
- **筋力訓練**（腓骨筋群、内在筋）
- **バランス・固有感覚訓練**（ボード、片脚立位、不安定面）
- **スポーツ特異的訓練**（agility, jump-landing）
- **装具療法**（lace-up brace, semirigid brace）— スポーツ復帰時の再受傷予防に有効

詳細プロトコルは [理学療法士向けページ](rehab.md) を参照。

---

## 4. 手術療法

### 4-1. 適応

- 適切な保存療法 **3–6か月** で症状改善が乏しい
- 反復する giving way による日常・スポーツ制限
- 合併病変（OLT等）が手術適応となる場合

### 4-2. 術式選択フローチャート

```
保存療法失敗
  └─ 良質な ATFL/CFL 残存組織あり？
        ├─ Yes → 解剖学的修復（Modified Broström-Gould）
        │        ・open / arthroscopic
        │        ・suture anchor 補強
        └─ No  → 解剖学的再建（autograft: 半腱様筋・薄筋 / allograft）
                  ・反復手術後、肥満、全身関節弛緩、ハイデマンド症例
```

非解剖学的再建（Watson-Jones, Evans, Chrisman-Snook）は腓骨筋腱を犠牲にし、距骨下関節可動域を制限するため **第一選択としては推奨されない**。

### 4-3. 標準術式：Modified Broström-Gould（観血法）

| ステップ | ポイント |
|---|---|
| 体位 | 仰臥位、患肢挙上、ターニケット使用 |
| 皮切 | 外果前縁に沿った J字 / 弧状切開（4–6 cm） |
| 神経注意 | **腓腹神経外側枝**（皮切後縁）、**浅腓骨神経**（皮切前縁）— 損傷で痛覚過敏 |
| 展開 | 伸筋支帯下を剥離、伸筋支帯（Gould 補強用）を温存 |
| ATFL 同定 | 前方関節包の肥厚層として確認、必要に応じて関節鏡で関節内処置を併施 |
| 縫合 | ATFL・前方関節包を **pants-over-vest** または **suture anchor**（腓骨側）で短縮縫合 |
| CFL 縫合 | 緩み著明なら同様に短縮縫合 |
| Gould 補強 | **伸筋支帯** を腓骨遠位骨膜へ重ね合わせ縫合 |
| 閉創 | 真皮縫合 + 表皮縫合、ドレーンは原則不要 |
| 外固定 | **短下肢シーネ（中間位）** 2週 → ブーツ装具 4–6 週 |

### 4-4. 関節鏡視下 Broström（Arthroscopic Broström）

- 関節内病変の同時処置が可能（OLT、滑膜切除、骨棘切除）
- 皮切が小さく整容性に優る
- **学習曲線あり**、神経損傷（浅腓骨神経・腓腹神経）に注意
- アウトカムは観血法と同等またはやや早期機能回復が報告

### 4-5. 解剖学的再建術

- グラフト: 半腱様筋（自家）、薄筋（自家）、足底筋、同種腱
- 腓骨・距骨・踵骨にトンネル作成 → 解剖学的に ATFL ± CFL を再建
- 適応: ATFL組織残存不良、反復術後、全身関節弛緩、肥満、高強度スポーツ復帰希望

### 4-6. 合併病変への併施手術

| 合併病変 | 対応 |
|----------|------|
| 骨軟骨損傷（OLT） | 関節鏡下マイクロフラクチャー、自家骨軟骨移植、BMS+生物製剤 |
| 後足部内反 | 踵骨外反骨切り術（Dwyer lateralizing calcaneal osteotomy） |
| 第1中足骨底屈 | dorsiflexion osteotomy of 1st metatarsal |
| 腓骨筋腱症・断裂 | デブリードマン、tubularization、腱移行 |
| 凹足（cavovarus） | アライメント補正術 |

## 5. 周術期管理・合併症

### 5-1. 主要合併症と頻度

| 合併症 | 頻度 | 対策 |
|--------|------|------|
| 浅腓骨神経・腓腹神経損傷 | 5–10% | 皮切設計、皮下展開を慎重に |
| 創部感染（SSI） | 1–3% | 喫煙・糖尿病・低栄養のコントロール |
| 深部静脈血栓症（DVT） | <2% | 早期足趾運動、リスク高ければ薬物予防 |
| 再不安定 | 5–15%（10年） | リハビリ完遂、再受傷予防教育 |
| 関節拘縮 | 5–10% | 早期可動域訓練 |
| Complex Regional Pain Syndrome（CRPS） | <2% | 早期発見、ペインクリニック介入 |

### 5-2. 術後プロトコル（標準）

| 時期 | 荷重 | 外固定 | リハビリ |
|------|------|--------|----------|
| 術後 0–2 週 | 非荷重 | 短下肢シーネ（中間位） | 足趾運動、患肢挙上、DVT予防 |
| 2–6 週 | 部分荷重 → 全荷重 | ブーツ装具 | 自動可動域、軽い腓骨筋等尺収縮 |
| 6–12 週 | 全荷重（装具内） | 装具 → ブレース | 抵抗運動、バランス訓練、固有感覚 |
| 3–6 か月 | フリー（スポーツ時はブレース） | スポーツブレース | アジリティ、ジャンプ着地、スポーツ復帰 |

詳細は [理学療法士向けページ](rehab.md) を参照。

### 5-3. 退院・外来フォロー

- 退院前: 創部状態、外固定の脱着、松葉杖歩行、シャワーの可否、危険サイン教育
- 外来: 術後 2 週（抜糸）、6 週（外固定除去）、3 か月、6 か月、1 年

---

## 6. アウトカム

- **Modified Broström-Gould** 法: 良好以上 85–95%、患者満足度 90% 前後
- スポーツ復帰率: 80–90%、復帰時期 中央値 4–6 か月
- 長期: 10年後の機械的不安定再発 約 10%、変形性関節症進行と相関
