# 足関節不安定症（外側側副靱帯損傷を含む）

!!! abstract "要点"
    - **急性外側側副靱帯損傷（捻挫）** と **慢性足関節不安定症（CAI）** は **同一スペクトラム** の連続病態
    - 主原因は **(a) スポーツによる急性外傷** と **(b) 加齢による靱帯変性** の2つ。どちらも放置すれば **スポーツパフォーマンス低下** → **変形性足関節症** へ進行
    - **慢性期は症状が微細で見逃されやすい**（漠然とした不安感、軽い痛み、giving way）
    - 当院標準術式は **関節鏡視下 Broström**。低侵襲・整容に優れ、関節内合併病変も同時処置可
    - 当院後療法プロトコル：**翌日から歩行許可** ／ 抜糸 **10–14日** ／ **包帯またはサポーター 6週間** ／ 入浴は **防水→抜糸後フリー** ／ **6週間で職場・スポーツ復帰**
