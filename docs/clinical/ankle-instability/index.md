# 足関節不安定症（Chronic Ankle Instability, CAI）

!!! abstract "要点"
    - 足関節捻挫の **20–40%** が慢性化し、慢性足関節不安定症（CAI）に至る
    - 病態は **機械的不安定性**（靱帯の緩み）と **機能的不安定性**（感覚運動制御障害）の混在
    - 初期治療は保存療法（3か月の運動療法）。改善が乏しければ手術を検討
    - 標準術式は **Modified Broström-Gould 法**。近年は **関節鏡視下手術**（arthroscopic Broström）が普及
    - 重度症例・反復術後症例では **解剖学的再建術（autograft/allograft tendon）** や **腓骨筋腱を用いた非解剖的再建** を検討
