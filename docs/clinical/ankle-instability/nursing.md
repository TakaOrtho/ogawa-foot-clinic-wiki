# 看護師向け：足関節不安定症の周術期看護

!!! abstract "看護師が押さえるポイント"
    1. 術前: **皮膚・末梢循環・喫煙・内服**の最終確認
    2. 術中持参物: **短下肢シーネ材料、患肢用枕、DVT予防**
    3. 術直後: **腓骨神経麻痺の有無**（母趾背屈・第1-2趾間感覚）を最優先に確認
    4. 病棟: **患肢挙上の徹底**、シーネ内浮腫・圧迫の観察
    5. 退院指導: **危険サイン**（強い痛み・しびれ・冷感・発熱）の即時連絡を教育

---

## 1. 術前看護

### 1-1. 入院時アセスメント

| 項目 | 観察・確認 |
|------|-----------|
| 皮膚 | 外果周囲の発赤・水疱・湿疹・白癬・潰瘍。**創部予定部位の異常は手術延期事由** |
| 末梢循環 | 足背動脈・後脛骨動脈触知、足趾の色調・温度、CRT |
| 神経学的所見 | 足趾運動、しびれ・感覚の左右差（既存所見の記録） |
| 既往・合併症 | 糖尿病（HbA1c）、抗血栓薬、ステロイド、関節リウマチ、末梢神経障害 |
| 内服 | 抗血栓薬の休薬指示確認、ステロイド・生物学的製剤の継続可否 |
| 生活習慣 | **喫煙**（創傷治癒・偽関節リスク）、飲酒、栄養状態 |
| 既往の足部外傷 | 反復捻挫の回数、最後の受傷時期、現在の症状程度 |

### 1-2. 患者教育（術前訪問・術前日）

- 手術の流れ（麻酔、所要時間、術後シーネ、ドレーン原則なし）
- **松葉杖の練習**（術後は非荷重 2 週間）
- 患肢挙上の重要性（心臓より高く、3点支持枕で 24h）
- 術後の疼痛コントロール方法（PCA・経口鎮痛薬）
- 禁飲食時間、内服薬の朝の指示
- 入院中の生活（シャワー、トイレ移動、着替え）

### 1-3. 術前準備（手術当日）

- 抗菌石鹸での下肢洗浄、足趾爪切り、爪垢除去
- マニキュア・ペディキュア除去（パルスオキシメーター用に複数指）
- アクセサリー類除去
- 弾性ストッキング装着（健側のみ、または両側 ※施設プロトコル）
- 抗菌薬投与時刻の確認

---

## 2. 術中・帰室時の引き継ぎ

### 2-1. 手術室からの申し送りで確認

- 術式（観血 / 関節鏡視下、追加術式の有無）
- 出血量、輸血の有無
- ターニケット使用時間
- ドレーン挿入の有無（通常なし、入っていれば部位・形式）
- 術中投与薬（抗菌薬、鎮痛薬、抗血栓薬）
- 関節内処置の有無（OLT処置 → 荷重制限変更の可能性）
- **外固定の種類**（シーネの角度、ブーツの有無）
- 神経ブロックの有無（→ 帰室後の感覚消失部位）

### 2-2. 帰室時バイタル

- 意識、呼吸、循環、SpO₂
- 創部出血量、シーネ内の出血浸潤の範囲（マーキング）
- 疼痛スケール（NRS / VAS）
- **足趾の運動・感覚・色調・冷感**（健側と比較）

---

## 3. 術後病棟管理

### 3-1. 観察ポイント（特に術後 24–48 時間）

!!! danger "緊急で医師報告すべき所見"
    - **足趾の強い痛み・しびれ・冷感・色調変化（蒼白・チアノーゼ）** → コンパートメント症候群、循環障害
    - **シーネ内の急速増大する腫脹**、シーネ圧迫感の訴え
    - **足趾運動消失**（特に母趾背屈不能 → 腓骨神経麻痺）
    - 安静時 NRS ≥ 7、鎮痛薬無効の強い痛み
    - 発熱 ≥ 38.0°C、創部発赤拡大、悪臭ある滲出
    - 呼吸困難・胸痛（→ PE 疑い）

| 観察項目 | 頻度（24h） | チェック内容 |
|----------|-------------|--------------|
| バイタル | 4 時間毎 | 体温・血圧・脈・SpO₂ |
| 末梢神経 | 2–4 時間毎 | 母趾背屈、第1-2趾間感覚（深腓骨神経）、足底感覚（脛骨神経）、足背感覚（浅腓骨神経） |
| 末梢循環 | 2–4 時間毎 | 足趾色調、温度、CRT、足背動脈 |
| 疼痛 | 各勤務 | NRS/VAS、鎮痛効果 |
| シーネ・創部 | 各勤務 | 出血量、浸潤、皮膚圧迫所見 |
| 患肢挙上 | 随時 | 心臓より高く、3点支持の維持 |

### 3-2. 神経学的観察の解剖

| 神経 | 評価部位 | 評価方法 |
|------|----------|----------|
| 深腓骨神経 | 第1-2趾間背側 | 触覚、母趾背屈筋力 |
| 浅腓骨神経 | 足背中央〜外側 | 触覚、足部外反筋力 |
| 脛骨神経 | 足底 | 触覚、足趾屈曲 |
| 腓腹神経 | 足部外側縁 | 触覚 |

!!! tip "実践のコツ"
    - **術前の所見を必ず記録**しておく（術後の左右差・術前後差を見るため）
    - 「しびれているか？」と聞くだけでなく、**触って答えてもらう** か、健側と比較させる
    - **腓骨神経麻痺は外側からのシーネ圧迫**でも起こる → シーネ巻き直しを医師と相談

### 3-3. 患肢挙上・浮腫管理

- 心臓よりも高い位置（枕 2–3 個、足挙上クッション）
- **足趾は枕外に出す**（趾の循環観察のため）
- 24 時間体位保持、ベッド上での体位変換時もキープ
- 初回起立・トイレ歩行は **必ず患肢挙上時間を確保した後**

### 3-4. 疼痛管理

- 定時鎮痛（NSAIDs / アセトアミノフェン）+ 頓用
- リハビリ前 30 分に予防的鎮痛
- 24h NRS が安定して 3 以下になることを目標
- 強オピオイドは便秘・悪心予防セットで

### 3-5. DVT 予防

- 術後早期からの **足趾運動（背屈・底屈）** を 1 時間に 10 回
- 健側の積極運動、間欠的空気圧迫
- 抗血栓薬投与時はリスク・出血徴候の観察
- ホーマンズ徴候、ふくらはぎの腫脹・熱感

### 3-6. 外固定（シーネ・ブーツ）管理

- **シーネのずれ・浸出**を毎勤務でチェック
- 患者が「きつい」と訴えたら必ず医師確認（自己判断で巻き直さない）
- 入浴・シャワー時は外固定を濡らさない（防水カバー使用）
- 装具に交換後は装着方法を再指導

---

## 4. リハビリ・ADL 援助

### 4-1. 初回起立・歩行

- 起立性低血圧予防（端座位 → 立位を段階的に）
- 松葉杖 / 歩行器の使用法を理学療法士と協働で再確認
- **非荷重歩行**（術後 2 週）— 患肢を完全に床から離す
- 階段昇降: 健側上り、患側下り（後述）

### 4-2. ADL 支援

| ADL | 留意点 |
|------|-------|
| トイレ | 患肢挙上できる多目的トイレ、ナースコール位置 |
| シャワー | 防水カバー、シャワーチェア、転倒予防マット |
| 着替え | 患側から着る／健側から脱ぐ（ゆとり優先） |
| 食事 | 患肢挙上したまま摂取できるテーブル配置 |
| 睡眠 | 患肢挙上枕、ナースコール |

### 4-3. 階段昇降の原則

- **上り**: 健側から上げる（"Good goes up"）
- **下り**: 患側から下ろす（"Bad goes down"）
- 松葉杖は必ず患側と同時

---

## 5. 退院指導

### 5-1. 退院前チェックリスト

- [ ] 創部・シーネの自己観察ができる
- [ ] 患肢挙上・松葉杖歩行が自立
- [ ] 鎮痛薬・抗血栓薬の自己管理可
- [ ] **危険サイン**を 5 つ言える（下記）
- [ ] 緊急連絡先を理解
- [ ] 次回外来日程を把握
- [ ] シャワーの可否と方法を理解
- [ ] 復職・スポーツ復帰の目安を理解

### 5-2. 患者・家族に伝える「危険サイン」

!!! warning "すぐに病院に連絡"
    1. **痛みが急に強くなる、薬が効かない**
    2. **足趾のしびれ・冷感・色が悪い**
    3. **シーネ内がきつくて痛い**
    4. **創部から膿・悪臭・赤みが広がる**
    5. **発熱（38°C以上）が続く**
    6. **ふくらはぎが腫れて痛い・赤い** → DVT
    7. **急な息切れ・胸痛** → 肺塞栓

### 5-3. 日常生活指導

- シャワー: 創部が乾燥・抜糸後（通常 2 週）まで防水カバー
- 入浴: 抜糸後 + 創部完全乾燥後
- 運転: 右足術側は装具除去・自由可動域回復後（おおむね 6–8 週）
- 飲酒・喫煙: **禁煙継続**を必ず伝える
- 食事: 高タンパク、便秘予防に水分・食物繊維

### 5-4. 外来フォロー連携

- 次回受診日、持参物（保険証、お薬手帳、シーネ等）
- 訪問看護・地域連携が必要なケース（高齢独居、移動困難）はソーシャルワーカーへ
